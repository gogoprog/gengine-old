gengine
=======

An experimental game engine

## Status
[![Build Status](https://drone.io/github.com/gogoprog/gengine/status.png)](https://drone.io/github.com/gogoprog/gengine/latest)

## Features
 * Emscripten/WebGL support
 * Lua scripting
 * Gameplay components
   * Sprite
   * Camera
   * Mouseable
   * Animated sprite
